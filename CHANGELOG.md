# 1.1.0

* Initial release of `power3d_annotations`.
* Provides premium annotation styles for the `power3d` plugin.
* Included Styles:
    * **Hotspot**: A clean, Sketchfab-style point-of-interest dot.
    * **TextCallout**: An interactive title-based annotation that expands to show details.
* Automatic style provisioning to the app's asset directory.
* Full integration with `Power3DController` via `initForAnnotations()`.
* **Documentation & Showcase**:
    * Added visual gallery showing hotspots and callouts.
    * Integrated with the core `power3d` engine's annotation system.
    * Comprehensive inline documentation and setup guide.
